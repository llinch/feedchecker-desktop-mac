# Информация о приложении

## Что такое Electron?

**Electron** — это фреймворк для создания десктопных приложений с помощью веб-технологий (HTML, CSS, JavaScript/TypeScript).

### Как это работает:

1. **Chromium** — встроенный браузер для отображения интерфейса
2. **Node.js** — для работы с файловой системой, процессами и системными API
3. **Main Process** — главный процесс, который управляет окнами и системными ресурсами
4. **Renderer Process** — процессы, которые отображают веб-контент (ваш React фронтенд)

### Преимущества:
- ✅ Один код для всех платформ (Windows, macOS, Linux)
- ✅ Использование знакомых веб-технологий
- ✅ Богатая экосистема npm пакетов
- ✅ Легкая разработка и поддержка

### Примеры приложений на Electron:
- Visual Studio Code
- Discord
- Slack
- WhatsApp Desktop
- Spotify

### В нашем приложении:
- **Main Process** (`main/index.js`) — управляет окнами, запускает Python бэкенд
- **Renderer Process** (`renderer/`) — React приложение с интерфейсом
- **Backend** (`backend/`) — Python FastAPI сервер для обработки фидов
