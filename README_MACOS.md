# Портативная версия для macOS

## Быстрый старт

### Если у вас есть Mac:

1. Установите зависимости (если еще не установлены):
   ```bash
   brew install node python@3.11
   ```

2. Установите npm зависимости:
   ```bash
   npm install
   cd renderer && npm install && cd ..
   ```

3. Соберите портативную версию:
   ```bash
   npm run build:portable:mac
   ```

4. Готово! Портативная версия в папке `portable/`

### Если у вас нет Mac:

**Варианты:**

1. **GitHub Actions** (бесплатно для публичных репозиториев)
   - Создайте workflow файл (см. `BUILD_MACOS.md`)
   - Запустите сборку в Actions
   - Скачайте артефакт

2. **Виртуальная машина macOS**
   - Установите macOS в VMware/VirtualBox
   - Соберите там

3. **Облачный Mac**
   - Используйте сервисы типа MacStadium или MacinCloud
   - Соберите там

4. **Попросите коллегу с Mac собрать**
   - Отправьте ему проект
   - Он запустит `npm run build:portable:mac`
   - Получите готовую папку `portable/`

## Использование

Пользователи просто копируют папку `portable/` и запускают `FeedChecker.app`.

**Примечание:** При первом запуске macOS может запросить разрешение. Нужно:
1. Правой кнопкой на `FeedChecker.app`
2. Выбрать "Открыть"
3. Подтвердить
